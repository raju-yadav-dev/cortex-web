What each package is best for

Windows EXE

Best for most desktop users

This is the simplest path if you want a guided installer that creates shortcuts and handles the app setup for you.

Windows MSI

Best for managed deployment

Use this package when you want a more enterprise-friendly installer for scripted or controlled Windows deployments.

Linux DEB

Best for Debian-based systems

Install the DEB package on Ubuntu, Debian, and similar distributions using your preferred package manager or the dpkg flow.
